# XGM Wellness Platform

## Developed by
**Everyday Plug**

## Project Overview

XGM Wellness Platform is a modern wellness e-commerce and reseller management system designed to help XGM Wellness sell products online while empowering independent resellers.

The platform will allow customers to browse products, place orders, track deliveries, and create secure accounts. It will also provide a reseller system, an admin dashboard, and order management tools.

---

## Features

- Customer Registration & Login
- Reseller Registration
- Product Catalogue
- Shopping Cart
- Secure Checkout
- Order Tracking
- Admin Dashboard
- Reseller Dashboard
- Pargo Shipping Integration
- Responsive Mobile Design
- WhatsApp Support
